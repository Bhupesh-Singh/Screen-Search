# Screen Search - Website

Professional website for the Screen Search Chrome Extension - Search Screen with Google Lens.

## Files Structure

- **index.html** - Main homepage with features, how it works, and download sections
- **privacy-policy.html** - Privacy policy page (required for Chrome Web Store listing)
- **styles.css** - All styling and responsive design
- **script.js** - Interactive features and smooth scrolling

## Features

✅ Modern, responsive design  
✅ Mobile-friendly layout  
✅ Privacy policy page (required for Chrome Web Store)  
✅ Smooth animations and transitions  
✅ SEO-friendly structure  
✅ Professional UI/UX  

## Setup Instructions

1. **Upload to your domain**: Upload all files to your web hosting service
2. **Update domain in Chrome Web Store**: 
   - Go to Chrome Web Store Developer Dashboard
   - Edit your extension listing
   - Add your website URL in the "Website" field
   - Add privacy policy URL (e.g., `https://yourdomain.com/privacy-policy.html`)

3. **Customization** (if needed):
   - Update email addresses if needed
   - Modify colors in `styles.css` (CSS variables at the top)
   - Update any specific content
